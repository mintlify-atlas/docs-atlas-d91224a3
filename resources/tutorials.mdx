## Getting Started

If you are a total newbie, try this guide:

- [Fuzzing Module](https://github.com/alex-maleno/Fuzzing-Module) - A comprehensive introduction to fuzzing for beginners

## Written Tutorials

Here are some excellent write-ups to show how to effectively use AFL++:

- [AFL++ libxml2 Tutorial](https://aflplus.plus/docs/tutorials/libxml2_tutorial/) - Official AFL++ tutorial using libxml2
- [Fuzzing a Game Boy Emulator](https://bananamafia.dev/post/gb-fuzz/) - Practical fuzzing of a complex application
- [The Art of Fuzzing](https://bushido-sec.com/index.php/2023/06/19/the-art-of-fuzzing/) - Comprehensive guide to fuzzing techniques

### GitHub Security Lab Series

- [Fuzzing Challenges and Solutions - Part 1](https://securitylab.github.com/research/fuzzing-challenges-solutions-1)
- [Fuzzing Software - Part 2](https://securitylab.github.com/research/fuzzing-software-2)
- [Fuzzing Sockets: FTP](https://securitylab.github.com/research/fuzzing-sockets-FTP)
- [Fuzzing Sockets: FreeRDP](https://securitylab.github.com/research/fuzzing-sockets-FreeRDP)
- [Fuzzing Apache - Part 1](https://securitylab.github.com/research/fuzzing-apache-1)

### Mode-Specific Tutorials

- [Fuzzing with QEMU Mode](https://airbus-seclab.github.io/AFLplusplus-blogpost/) - Airbus SecLab's guide to binary-only fuzzing
- [Android Fuzzing with Frida Mode](https://blog.quarkslab.com/android-greybox-fuzzing-with-afl-frida-mode.html) - Quarkslab's guide to fuzzing Android apps
- [Fuzzing Teeworlds Map Parser](https://mmmds.pl/fuzzing-map-parser-part-1-teeworlds/) - Real-world fuzzing example

## Structured Data Fuzzing

If you are interested in fuzzing structured data (where you define what the structure is):

- [libprotobuf for AFL++](https://github.com/P1umer/AFLplusplus-protobuf-mutator)
- [libprotobuf raw implementation](https://github.com/bruce30262/libprotobuf-mutator_fuzzing_learning/tree/master/4_libprotobuf_aflpp_custom_mutator)
- [libprotobuf for old AFL++ API](https://github.com/thebabush/afl-libprotobuf-mutator)
- [Superion for AFL++](https://github.com/adrian-rt/superion-mutator) - Grammar-based fuzzing
